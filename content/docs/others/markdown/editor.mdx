常用的 Markdown 编辑器有以下几个：

- [**VSCode**](https://code.visualstudio.com/) ：主播倒是比较建议使用 VSCode 编写 md 文档(虽然我正在使用 Obsidian)。用 VSCode 编写 Markdown 可以让你更加了解每一个符号的用法。
  <Callout>**Tips:** 可以使用 VSCode 中强大的插件来帮助你编写 Markdown 哦！</Callout>
- [**Typora**](https://typora.io/) ：简洁，功能全面，但是收钱(虽然盗版泛滥，但是主播不建议使用盗版😳，**请尊重人家的劳动成果与版权**)。
- [**Joplin**](https://joplinapp.org/) ：免费，开源，插件较少，功能基本满足所需。
- [**Obsidian**](https://obsidian.md/) ：插件多，但是过于冗杂，同时有付费的官方的同步和自建或第三方(比如 iCloud、Git)的同步方式。
