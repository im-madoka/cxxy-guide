Markdown 可以说是计算机学生前期必学的语言，他具有着易读易写的特点，通常用于编写博客、技术文档、使用手册等，也可以作为学习路程上记录笔记的方式。

在软件开发领域，Markdown 已成为技术文档的标准格式。

这篇教程就是由 (带有拓展的) Markdown 编写。
