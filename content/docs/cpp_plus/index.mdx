## 欢迎来到 C++ Plus

如果你已经阅读完 [C++ 基础](/docs/cpp) 章节，掌握了变量、循环、指针和类的基本用法，那么恭喜你，你已经拿到了通往 C++ 世界的入场券。

但是，在写代码的过程中，你是否会有这样的困惑：

- 为什么我的 C++ 代码看起来和 C 语言没什么两样，只是多了个 `class`？（俗称 "C with Classes"）
- 什么是“现代 C++”？`auto`、智能指针、右值引用到底在解决什么问题？
- 代码写多了全堆在一个 `main.cpp` 里，想找代码都找不到，怎么办？

如果这些问题曾困扰过你，那么这份 **C++ Plus 进阶指南** 就是为你准备的。

## 在这份指南里我能学到什么？

基础篇教你**“怎么写出能跑的代码”**，而进阶篇将教你**“怎么写出优雅、健壮、可维护的工程级代码”**。

我们将不再纠结于分号丢哪里的语法细节，而是将目光投向**现代化**、**工程化**和**抽象化**。

## 准备好了吗？

进阶之路或许比入门更加陡峭，因为你需要开始思考“为什么”而不仅仅是“是什么”。但请相信，当你站在更高处俯瞰时，风景必定更加壮丽。

> "C++ is designed to allow you to express ideas, not just write code." —— Bjarne Stroustrup

让我们开始这段全新的旅程吧！
