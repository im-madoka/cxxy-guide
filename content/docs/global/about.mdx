这是由成贤计协学生创建并维护的知识库，旨在帮助大家更好的学习和掌握计算机领域有关的知识。

如果你想要对本仓库进行贡献，你可以将你写的文档发送至邮箱 [im.leojia@gmail.com](mailto:im.leojia@gmail.com) 或者前往 [指南仓库](https://github.com/im-madoka/cxxy-guide) 提交 PR。

本知识库还在持续施工中，目前计划如下：

- [x] C/C++ 入门指南 (100/100%)
- [ ] C++ 进阶指南 (10/100%)
- [ ] JavaScript/TypeScript 入门指南 (0/100%)
- [ ] 杂项（Git/Markdown/Linux/Shell）

如果对文档的内容有任何疑问，或是在编程上有任何 **无法自行解决的** 需要帮助的地方，欢迎在计协 QQ 群中提问。
